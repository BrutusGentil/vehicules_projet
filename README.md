# vehicules\_projet

Projeto do Sprint 5

**Descrição do Projeto**
Este projeto faz parte da Sprint 5 do curso de Analista de Dados da TripleTen.
O objetivo é criar uma aplicação web interativa utilizando Streamlit, permitindo visualizar e explorar dados de veículos usados através de gráficos no caso histogramas e gráficos de dispersão

A aplicação está publicada online através do serviço Render.

**Funcionalidades do Aplicativo**
A aplicação oferece quatro visualizações principais:

1. Histograma do preço dos veículos
Ativado por botão.
Permite observar a distribuição dos preços no dataset.
2. Gráfico de dispersão (model\_year × price)
Ativado por botão.
Mostra a relação entre o ano do modelo e o preço do veículo.
3. Histograma adicional (via checkbox)
Ativado por caixa de seleção.
Permite visualizar outra distribuição (no caso a kilometragem das viaturas).
4. Gráfico de dispersão adicional (via checkbox)
Ativado por caixa de seleção.
Mostra outra relação entre variáveis: o preço e os dias que o veículo está listado.
